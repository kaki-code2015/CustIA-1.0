## 💡 À propos
**CustIA** est un projet léger qui permet la création de fichiers personnalisés utilisant l'extension `.NayMOD`. L'objectif est de proposer une solution simple, sans fioritures, pour structurer tes données.

## ✨ Fonctionnalités
* ✅ **Génération de fichiers .NayMOD** : Crée tes fichiers en un clin d'œil.
* ✅ **Zéro dépendance** : Fonctionne avec l'installation de base de Python.
* ✅ **Léger & Rapide** : Pas de bibliothèques lourdes à charger.
